# AI-Powered Autonomous Medicine Delivery Robot

An autonomous robot that identifies, picks, and delivers medicines using real-time computer vision and sensor fusion — built to assist hospital staff with routine medicine delivery.

## 🎯 Problem Statement
Hospital staff spend significant time on repetitive medicine delivery rounds. This project automates that process using AI-driven object detection and autonomous navigation, reducing staff workload and response time.

## 🛠️ Tech Stack
- **Vision:** YOLO (real-time object detection)
- **Hardware:** Raspberry Pi, RFID module, Ultrasonic & IR sensors
- **Connectivity:** IoT integration, mobile app monitoring
- **Language:** Python

## ⚙️ How It Works
1. YOLO-based detection pipeline identifies medicine packets across 15+ drug labels under varying lighting conditions
2. RFID-based navigation combined with ultrasonic + IR sensor fusion enables autonomous corridor traversal
3. IoT module sends live status updates to a connected mobile app for remote monitoring

## 📊 Results
- Reliable medicine identification across 15+ drug labels
- Zero manual interventions required during autonomous delivery trials
- ~40% reduction in estimated nurse response time for routine deliveries
